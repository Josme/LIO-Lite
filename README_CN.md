# LIO-Lite
一个可以用在无人机上的轻量级 LIO 系统：适配了Livox Mid-360雷达。  
整个项目是在[lio-sam](https://github.com/TixiaoShan/LIO-SAM.git)的基础上做了一些优化  


## Illustrate
c++ == 17  
gtsam == 4.3  
项目整体使用C++17的标准   
gtsam需要自己编译，放在了3rdparty中：提供了x86和arm两个版本   
当然，你也可以自己编译替换掉
